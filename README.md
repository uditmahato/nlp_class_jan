## ğŸš€ Getting Started

### Prerequisites
- Git installed on your computer
- A GitHub account
- Basic command line knowledge

### First Time Setup

```bash
# Fork the repository on GitHub first, then clone your fork
git clone https://github.com/YOUR-USERNAME/nlp_class_jan.git
cd nlp_class_jan

# Create your feature branch
git checkout -b your-name-first-contribution

# Make your changes (edit your assigned student file)

# Stage and commit
git add .
git commit -m "Add [YourName]'s profile"

# Push to GitHub
git push origin your-name-first-contribution
```

Then create a Pull Request on GitHub!
